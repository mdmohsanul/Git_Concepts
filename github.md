# What is a Pull Request (PR) and why is it used?
A pull request is a request to merge your branch into another branch (usually main or develop) via platforms like GitHub/GitLab/Bitbucket.
Why used
Code reviews
Discuss changes
Run CI/CD tests
Maintain clean and safe merges
Ensure team collaboration

PR = quality gate before merging code.
